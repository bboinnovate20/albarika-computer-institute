# CSS PRACTICE EXERCISES

This assignment is designed to help you practice **common CSS properties**, including:
- font size
- margin and padding
- border
- text color
- background color
- basic list styling

You are required to **write only CSS**.  
The HTML structure will be provided separately.

---

## EXERCISE 1: Profile Card Styling

### Objective
Style a profile card that contains text, a button, and an unordered list of skills.

### Instructions

1. Style the page body
   - Set a background color `#f2f2f2`
   - Use Arial or a sans-serif font

2. Style the container class
   - Set a fixed width
   - Center it horizontally on the page
   - Add padding inside the card
   - Set a white background color
   - Add a solid border
   - Round the corners slightly

3. Style the main heading
   - Increase the font size
   - Change the text color `#333333`

4. Style the subtitle text
   - Use a smaller font size than the heading
   - Set the text color to gray `#777777`

5. Style the description paragraph
   - Set a readable font size to `16px`
   - Add space above and below the text

6. Style the skills list
   - Remove the default list style
   - Remove the default left padding
   - List item text color: `#444444`
   - Add space between each list item

7. Style the button
   - Add padding
   - Set a background color `#007bff`
   - Remove the default border
   - Change the text color to white
   - Change the cursor to pointer on hover

## EXERCISE 2: Blog Page Styling

### Objective
Style a simple blog page that contains multiple posts and an ordered list.

### Instructions

1. Style the page body
   - Set a background color  `#e6f0ff`
   - Use a readable font - 16px
   - Remove the default browser margin

2. Style the main container
   - Set the width to about 70% of the page
   - Center it using margin
   - Add padding inside the container
   - Set the background color to white

3. Style each blog post
   - Add a solid border
   - Add padding inside each post
   - Add space between posts
   - Use a light background color `#fafafa`

4. Style the main page heading
   - Center align the text
   - Increase the font size
   - Text Color `#222222`

5. Style post headings
   - Change the text color
   - Add space below the heading

6. Style post date text
   - Use a smaller font size
   - Set the text color `#0056b3`

7. Style the ordered list
   - Add left margin
   - Increase spacing between list items

8. Style all paragraphs
   - Set the font size to 16px
   - Increase line height for readability
   - Set text color `#444444`


- Add rounded corners to cards and posts
- Add a hover effect to buttons
- Add a shadow effect to improve visual depth
